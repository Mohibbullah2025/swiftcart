1/ What is the difference between null and undefined?

Undefined মানে হলো কোনো vairable লেখা হয়েছে কিন্তা তার কোনো মান দেওয়া হয়নি এমন variable. আর null যদি আমরা ইচ্ছা করে কোনো variable এর মান কিছু নেই হিসাবে সেট করি তবে তাকে null বলে। 

2/ What is the use of the map() function in JavaScript? How is it different from forEach()?

কোনো array থেকে প্রতিটা element নিয়ে তার উপর কোনো function apply নতুন একটা array তৈরি করার জন্য map() use করা হয়। 
map() আবার নতুন একটা array return করে কিন্তু forEach() কোনো array return করে না। এই দিক থেকে map() forEach() থেকে ভিন্ন। 

3/ What is the difference between == and ===?

== দিয়ে শুধুমাত্র মান চেক করা হয় কিন্তু === দিয়ে মান এর সাথে সাথে type ও চেক করা হয়।

4/ What is the significance of async/await in fetching API data?

 aysynchronous code কে  synchronous code এর মতো করে লিখতে aysnc/await use করা হয়। আর await use করলে app call করার সময় response এর জন্য অপেক্ষা করে তারপরে পরের ধাপে যাই। 

 5/ Explain the concept of Scope in JavaScript (Global, Function, Block).

JavaScript এ Scope ৩ ধরনের। Global, Function এবং Block

Global Scope কে যেকোনো যাইগা থেকে access করা যাই। 
Function Scope কে শুধুমাত্র function এর ভিতর থেকে access করা যাই। 
আর  {} এর ভিতরের scope কে block scope বলে।
